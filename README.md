# Sentence Completion

Tiny app which suggests the next word based on whatever was typed last.

## Contributers

- [Holytheif](https://github.com/Holytheif/)
- [therisingtsun](https://github.com/therisingtsun)

---

## Installation

Clone the repository and run `npm install` (Requires [Node.js](https://nodejs.org/)).

## Running

Run `npm start`.
